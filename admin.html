Firebase Setup Guide

To fully activate Firebase, complete these steps:

  1. Go to console.firebase.google.com → Create project → "airskylink-travels"
  2. Enable Authentication → Sign-in method → Email/Password → Enable
  3. Create your admin account under Authentication → Users → Add user
  4. Enable Firestore Database → Start in production mode
  5. Go to Project Settings → Your apps → Web → Copy config
  6. Paste config in FB_CONFIG in both airskylink.html and admin.html
  7. Set Firestore Rules (copy from README or ask Harsh)
